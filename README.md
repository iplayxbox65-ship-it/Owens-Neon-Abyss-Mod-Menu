> **Platform:** PC (Steam)
> **Game:** Neon Abyss
> **Intended Use:** Offline / Single-Player Only

## Installation

### Requirements

* Neon Abyss (Steam version)

### Install Steps

1. Download the latest release from the GitHub Releases page.
2. Extract the ZIP file directly into your **Neon Abyss** installation directory.
3. Ensure all files and folders from the release are copied into the game directory.
4. Launch the game normally.
5. Open the mod menu using the configured hotkey.

### Example Directory

```text
Neon Abyss/
├── Mods/
├── plugins/
├── config/
├── [additional files from the ZIP]
└── Neon Abyss.exe
```

> **Important:** The release ZIP includes the required mod loader components and dependencies. Install the full package as provided.
